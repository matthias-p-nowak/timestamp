# Examples

### Instructions
- all examples are stored under the `examples` folder.
- use `examples/mobile.html` as the editable working file.
- when a version is approved, copy it to `examples/snapshots/mobile-YYYY-MM-DD-vN.html`.
- register each approved copy in `examples/snapshots/README.md`.

## `examples/mobile.html`
- contains a full html page with example made-up data

## `examples/snapshots/`
- contains dated snapshot files of approved `mobile.html` versions.

## `examples/calendar.html`
- contains a html page with a date picker
